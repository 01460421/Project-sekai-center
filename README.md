# Project SEKAI 資源中心

プロジェクトセカイ カラフルステージ！ feat. 初音ミク

專業論文級風格的資源整理網站，收錄官方資源、資訊站、討論社群、Wiki 百科等連結。

## 特色

- 📚 **論文級排版**：簡潔專業的設計風格，宋體 + Times New Roman
- 🔍 **即時搜尋**：快速篩選所需資源
- 📱 **全平台響應式**：桌面、平板、手機完美適配
- 🌓 **深色模式**：自動跟隨系統主題
- ⚡ **靜態網站**：載入快速，無需後端
- 📊 **嵌入式卡池**：直接顯示 Google Sheets 卡池列表

## 部署

### Vercel 部署（推薦）

1. Fork 此 Repository
2. 前往 [Vercel](https://vercel.com) 並登入
3. 點選 "New Project"
4. 選擇你 Fork 的 Repository
5. 點選 "Deploy"

### GitHub Pages 部署

1. 前往 Repository Settings
2. 選擇 Pages
3. Source 選擇 `main` branch
4. 儲存後等待部署完成
