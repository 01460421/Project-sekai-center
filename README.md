# Project SEKAI 資源中心

## 功能

- 📅 **卡池行事曆** - 月曆視圖顯示卡池期間，點擊查看詳情
- 📊 **卡池列表** - 台服卡池時間表（2025/12/31起）
- 🎵 **歌曲清單** - V+ P主列表 & 世界計畫收錄曲（含 YouTube 連結）
- 🧮 **推隊倍率計算器** - 公式：[隊長% + 100% + (隊員%總和÷5)] ÷ 100%
- 📱 **全平台響應式** - 桌面、平板、手機完美適配
- 🌓 **深色模式** - 自動跟隨系統主題

## 行事曆功能

- 月曆視圖顯示所有進行中的卡池
- 顏色標示不同卡池類型
- 點擊卡池查看詳細資訊
- 支援上/下月切換、快速跳轉今天

## 部署

1. 上傳至 GitHub
2. Settings → Pages → Source: main branch
3. 或連接 Vercel 自動部署
