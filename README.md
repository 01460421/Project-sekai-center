# Project SEKAI 資源中心

世界計畫 彩色舞台 feat. 初音未來 資訊整合網站

## 功能

- 📅 活動日曆 - 卡池開放時間視覺化
- 🎰 卡池列表 - 台服卡池時間表
- 🎵 歌曲資料庫 - 完整樂曲清單含：
  - 時間性質（翻唱/書下/收錄）
  - 團體分類篩選
  - 全難度譜面資訊（Easy ~ APPEND）
  - 排序與篩選功能
- 🔢 推隊倍率計算器

## 資料來源

- 歌曲資料：[Sekai.best API](https://sekai.best/)
- 卡池資料：手動整理

## 部署

```bash
npm install
npm run dev
```

或直接開啟 `index.html`
